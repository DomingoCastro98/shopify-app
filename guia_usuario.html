Shopify Utilidades App

Tu asistente gráfico definitivo para gestionar entornos de desarrollo locales de Shopify vía Docker, sin dependencias locales globales.

¿Qué hace esta App?

Desarrollar temas en Shopify normalmente requiere instalar Node.js, Ruby y la CLI de Shopify a nivel global en tu ordenador. Esta aplicación gráfica en Python elimina esa necesidad al encapsular todo tu entorno en contenedores de Docker.

Toda la comunicación de comandos viaja entre la app (GUI) y el contenedor seleccionado, permitiéndote tener versiones aisladas y seguras de tu código de producción listos para editar localmente.

Crear y Recrear Entorno

Crear: Inicializa un contenedor Docker nuevo desde cero. Descarga de forma segura la herramienta Shopify CLI internamente, te pide sincronizar la URL de tu tienda, realiza un clon del entorno remoto de la tienda y ejecuta automáticamente las descargas y rutinas NPM.

Recrear: Si sientes que el entorno está corrupto (errores NPM, problemas de permisos o bloqueos en Shopify CLI), esta opción borra totalmente el contenedor antiguo, recrea una versión limpia de cero basándose en el repositorio original o clon del tema remoto y restablece las dependencias de forma segura.

Importar Theme y Datos

Transfiere fácilmente código desde tu ordenador (un ZIP o un directorio) hacia dentro del contenedor virtual de Docker.

  • Copilado y npm install automáticos.
  • Proceso de AutoFix Inteligente: La aplicación lee settings_schema.json y repara silenciosamente discrepancias e incongruencias en el fichero de datos settings_data.json (ej. números donde la tienda remota obliga formato string o bordes que exceden esquemas).
  • Soporta despliegue ('Push') automático seguro a la tienda Shopify en Real (Live).

Exportar Theme y Datos

El asistente más importante para hacer un backup o extraer tu trabajo antes de subirlo.

  • Exportación Local: Empaqueta en formato .tar el entorno completo /app en Docker para extraerlo a tu escritorio/documentos de forma rápida y persistente (con todos tus cambios no guardados en remoto).
  • Descarga Remota ('Pull'): La aplicación abre de forma segura una CLI de Shopify en una consola embebida que descarga a la fuerza (vía credencial temporal) todos los assets directamente desde remoto e introduce estos ficheros dentro del paquete de exportación local descargado.

Gestión de Contenedores y Mercados

Desde la barra de navegación principal de la izquierda de la app, manejas la 'Sala de Máquinas':

  • Iniciar/Parar y eliminar entornos de trabajo.
  • Saltar a los ficheros Log de Docker o abrir una Shell interactiva directamente al entorno del contenedor para debugear comandos sin salir a consola o configurar entornos multi URL.
  • Informes, estadísticas de mercado de Shopify CLI e información local centralizada de la base de datos de Docker.
